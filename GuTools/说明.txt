咕咕牛图库 Web 管理器 - 部署说明

1. 运行时环境
Node.js: v16.x LTS 或更高版本。
Git: 系统 PATH 环境变量中必须包含 Git 可执行文件。
包管理器: 支持 npm 或 pnpm。

2. 依赖注入
进入 GuTools 源码根目录，执行以下命令注入生产环境依赖。

npm install --prod 或者   pnpm install --prod
注：sharp 库包含原生模块编译，首次注入耗时较长属正常现象。若遇 EACCES 权限错误，请提权到管理员运行终端。

3. 服务启动
依赖注入完成后，启动 Node.js 主进程。

node server.js

服务进程成功引导后，标准输出将显示监听地址及端口。
